## Python Algorithms Library
## Laurent Luce

### Description
The purpose of this library is to help you with common algorithms like:

String matching
  - Naive
  - Rabin-Karp
  - Knuth-Morris-Pratt
  - Boyer-Moore-Horspool

Binary tree
  - node and tree class
  - lookup
  - insert
  - delete
  - compare 2 trees
  - print tree
  - tree inorder generator

### Installation
Get the source and run

    $ python setup.py install
